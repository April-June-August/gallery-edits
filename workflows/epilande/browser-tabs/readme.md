## Usage

Search for pages in open browser tabs via the respective keywords:

* `safari tabs`
* `chrome tabs`
* `brave tabs`
* `vivaldi tabs`
* `edge tabs`
* `orion tabs`
* `arc tabs`

![Alfred search for brave tabs alfred](images/brave.png)

* <kbd>↩</kbd> Switch to the tab.
* <kbd>⌃</kbd><kbd>↩</kbd> Copy tab URL to the clipboard.
