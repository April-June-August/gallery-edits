## Usage

Interact with the Arc browser via the workflow's keywords:

* `atabs` Search all tabs.
* `bms` Search pinned tabs.
* `tabs` Search unpinned tabs.
* `tops` Search favorite tabs.
* `spaces` Search spaces.
